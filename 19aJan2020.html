  XD Plugin APIs: Drag-and-Drop and Stars  
 

As we kick off the new year, there are some exciting updates in the January release for XD including Content-Aware Layout and enhanced XD Plugin APIs. 
 
We’ve updated the XD plugin APIs to support drag-and-drop interactions, such as dragging an icon from a panel plugin onto the canvas. Also new is the Polygon.starRatio property and an experimental advanced debugger.
 
If you want to build an XD plugin or have an existing XD plugin you want to update, you can check out this sample code for adding drag-and-drop functionality to your plugin here.

  Instantly Generate Icons for Your XD Plugin  
 

We’re back with a new template that helps you generate icons for your XD plugins. This is a Part II to our UI Kits and UX Guidelines sticker sheet that we shared for MAX 2019.  
 
This template is a great resource for XD plugin developers and/or designers who are ready to start distributing an Adobe XD plugin in the Plugin Manager. Get set up with what you need today!
